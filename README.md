# Game-Theoretic-Os
# A Game-Theoretic Model of Strategic Interaction in Operating Systems

## Overview
This project explores CPU scheduling in operating systems using game theory. Processes are modeled as strategic agents competing for CPU resources.

## Objectives
- Model CPU scheduling as a strategic game
- Analyze Nash equilibrium behavior
- Compute Price of Anarchy
- Design incentive-compatible scheduling mechanisms
- Simulate and compare with classical schedulers

## Concepts Used
- Operating Systems (CPU Scheduling)
- Game Theory (Nash Equilibrium, PoA)
- Mechanism Design
- Simulation (C++)
